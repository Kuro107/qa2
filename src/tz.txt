Программа для работы с БД авиакомпании
У нас есть авиакомпания, у нее есть рейсы, клиенты и билеты.
Ваша цель, разработать консольную программу для добавления, удаления, изменения, фильтрации и поиска данных в БД.
Первая задача состоит в создании таблиц для будущей БД (все скрипты созданных таблиц скопировать в файл .txt или .sql и сохранить эти файлы в отдельной папке sql-skripts в проекте для проверки).
Нужно создать таблицу:
 Рейсов, которые должны содержать информацию о модели самолета, времени вылета, откуда вылет (код аэропорта), где будет прилет (код аэропорта),
 длительности полета, количество мест и номер рейса (2б)
 Стран, которые будут содержать код страны и ее наименование (1б)
 Аэропортов, которые будут содержать код аэропорта, город в котором находится и код страны (1б)
 Клиентов, которые должны содержать информацию по типу паспортных данных (инн, айди паспорта, ФИО, пол, страна и тд) (1б)
 Билетов, которые будут содержать в себе айди клиента, номер рейса, время получения билета и номер билета (2б)
